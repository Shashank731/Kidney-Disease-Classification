# Kidney Disease Classification

A Deep Learning project for kidney disease classification.

---

## Installation

### Clone the repository

```bash
git clone https://github.com/Shashank731/Kidney-Disease-Classification.git
cd Kidney-Disease-Classification
```

### Create environment

```bash
conda create -n kidney_env python=3.10 -y
conda activate kidney_env
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

## Usage

### Run training

```bash
python src/kidney_disease_prediction/pipeline/training_pipeline.py
```

### Run app

```bash
python app.py
```

Open browser:
http://localhost:8080
